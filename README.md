# Graduate Software Engineer Toolkit

A collection of practical templates, guides and resources created to help Computer Science and Software Engineering students prepare for graduate software engineering roles.

## What's Included

- ATS-Friendly CV Toolkit
- LinkedIn Optimisation Guide
- GitHub Portfolio Guide
- README Templates
- Project Presentation Toolkit
- Interview Preparation Guide
- Job Application Tracker
- AI Prompt Library
- Bonus Resources

## Who It's For

- Computer Science students
- Software Engineering students
- Recent graduates
- Internship applicants
- Junior software developers

## Why I Created It

During my final year studying Software Engineering, I found myself repeatedly improving my CV, GitHub portfolio, LinkedIn profile and interview preparation while applying for graduate roles.

I created this toolkit to organise everything I learned into one place so that other students can spend less time searching for advice and more time building their portfolio.
